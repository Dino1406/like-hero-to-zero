# Contributing

Danke, dass du beitragen möchtest! Kleiner Leitfaden:

1. Fork das Repo.
2. Erstelle einen neuen Branch: `git checkout -b feature/mein-feature`
3. Mach deine Änderungen, schreibe Tests.
4. Committe mit klaren Nachrichten: `git commit -m "feat: kurze Beschreibung"`
5. Push in deinen Fork und öffne eine Pull Request.

Bitte halte dich an den vorhandenen Code‑Style und ergänze Tests für neue Funktionalität.
